### thread
> ``named fun thread``

#### Описание
Создаёт и запускает новый поток.

#### Пример использования
```
var str = "test";
thread {
    str = "123";
}

thread -> println(str);
```

Выведет:
```
test
```
либо
```
123
```
Зависит от скорости выполнения
